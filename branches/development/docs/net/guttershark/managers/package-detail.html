Package net.guttershark.managersClasses
 

The net.guttershark.managers package contains classes for managing player wide functionality.


Classes
 ClassDescription
 AssetManager The AssetManager Class is a singleton that stores all assets loaded by any PreloadController.
 EventManager The EventManager class simplifies events and provides shortcuts for event listeners for numerous AS3 top level classes, guttershark classes, and component events on an opt-in basis.
 FormManager The FormManager Class simplifies managing InteractiveObjects that compose a form - THIS IS STILL IN DEVELOPMENT, BUT EVERYTHING HERE WORKS, IT'S JUST NOT FEATURE COMPLETE.
 KeyboardEventManager The KeyboardEventManager class simplifies working with keyboard events.
 LanguageManager The LanguageManager manages loading different language xml files, and handling updating your text fields with different text for a different language.
 LayoutManager The LayoutManager Class adds support for "layer" management with a display objects children.
 PlayerManager The PlayerManager Class provides static shortcut methods for finding things about the currently running player.
 ServiceManager The ServiceManager Class supports making Remoting requests, and normal HTTP service requests, with support for retries, timeouts, and some features that are specific to one or the other.
 SoundManager The SoundManager Class is used to control sounds document wide.
