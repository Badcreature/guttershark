Class FrameDelayMethods
 
Packagenet.guttershark.util
Classpublic class FrameDelay

The FrameDelay class allows for a callback to be called after a certain amount of frames have passed.


Example
Using a FrameDelay: 
 
  var fd:FrameDelay = new FrameDelay(after10, 10);
  function after10():void
  {
      trace("10 frames have passed");
  }
