Classpublic class CPU

The CPU class benchmarks the clients computer and provide a number you can use to decide if their cpu is fast / med / slow, etc.

The CPU class does not provide details about how many CPU's they have, or RAM information.

The recommended number ranges for fast / med / slow:

  • fast = (CPU.Speed < 45)
  • med = (CPU.Speed < 80)
  • slow = anything greater than the medium benchmark.

This class is integrated with the base DocumentClass of the com.mccannsf package.


Example
Benchmarking client CPU: 
 
  CPU.calculate();
  trace(CPU.Speed);
  trace(CPU.Speed == CPU.FAST);
  trace(CPU.Speed == CPU.MEDIUM);
  trace(CPU.Speed == CPU.LOW);
  trace(CPU.Benchmark);
