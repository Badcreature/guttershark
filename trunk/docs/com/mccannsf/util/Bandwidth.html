The Bandwidth class is used for sniffing the clients bandwidth.

This class is integrated with the base DocumentController of the com.mccannsf package.


Example
Sniffing the users' bandwidth. 
 
     var bs:Bandwidth = new Bandwidth();
     bs.contentLoader.addEventListener(Event.COMPLETE, onComplete);
     bs.detect();
     function onComplete(e:Event)
     {
       trace(Bandwidth.Speed);
       //Bandwidth.Speed will be equal to Bandwidth.LOW / MED / HIGH. One of the three.
     }

Bandwidth()constructor
public function Bandwidth(image:URLRequest = null, targetLowBandwidth:Number = 256, targetMedBandwidth:Number = 550)

Constructor for Bandwidth instances.

Parameters
image:URLRequest (default = null) — This is an optional parameter. By default the sniffer loads a ./bandwidth.jpg file. You can pass a different URLRequest here to change what file is used in the bandwidth detection.
 
targetLowBandwidth:Number (default = 256) — The target Kbps for low bandwidth
 
targetMedBandwidth:Number (default = 550) — The target Kbps for med bandwidth

The target high bandwidth is not a parameter because anything higher than the target medium bandwidth Kbps will automatically be high.
