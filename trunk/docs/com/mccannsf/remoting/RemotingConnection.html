Packagecom.mccannsf.remoting
Classpublic class RemotingConnection
InheritanceRemotingConnection flash.events.EventDispatcher

The RemotingConnection class simplifies connecting a net connection to a Flash Remoting gateway.

The RemotingConnection class must be used to connect to the gateway, and then given to a RemotingService instance.
