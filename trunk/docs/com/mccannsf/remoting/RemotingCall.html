Classpublic class RemotingCall
InheritanceRemotingCall flash.events.EventDispatcher

The RemotingCall class encapsulates a remoting request, and manages timing tasks with the call.

The RemotingCall class is used internally to a RemotingService and should not be used directly.

See also

public function RemotingCall(remotingService:RemotingService, method:String, onResult:Function, onFault:Function, args:Array, returnArgs:Boolean, callTimeout:int = 30000, maxRetries:int = 3)

New remoting call. This is not meant to be used outside of a RemotingService.

Parameters
remotingService:RemotingService — The RemotingService used for this call.
 
method:String — The operation to call on the connection.
 
onResult:Function — The result function to call.
 
onFault:Function — The fault function to call.
 
args:Array — The arguments to send to the net connection.
 
returnArgs:Boolean — Whether or not to send the original arguments to the result or fault callback.
 
callTimeout:int (default = 30000) — The time before a retry occurs in milliseconds.
 
maxRetries:int (default = 3) — The maximum number of retries.

Throws
— If the remoting service was null.
 
— If the method was null.
