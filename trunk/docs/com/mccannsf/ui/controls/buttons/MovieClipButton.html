The MovieClipButton class is a generic class you can bind any movie clip to in the Library, and get canned button states.

A frame in the movie clip represents a state that the button is in.

Default frame states:

  • 1 = normal state
  • 2 = over state
  • 3 = down state
  • 4 = locked frame. Toggle the lock state by setting the
  • locked
    • property.

Sound support is also available for each of the button states.

Sound Properties

  • overSound
  • downSound
  • upSound
  • outSound
  • lockedSound
  • unlockedSound

This class also uses high priority event listeners for mouse events. When you set the locked property, all MouseEvents will not be dispatched.

If you absolutely need mouse events even though the button is locked, you can add an event listener with a higher priority than 10.
