Packagecom.mccannsf.ui.controls.buttons
Classpublic class MovieClipCheckBox
InheritanceMovieClipCheckBox MovieClipButton flash.display.MovieClip

The MovieClipCheckBox class is a basic checkbox that extends from MovieClipButton and adds more functionality to control the "checked" state with frames on the movie clip.

Default frame states:

  • 1 = normal state
  • 2 = over state
  • 3 = checked normal state
  • 4 = checked over state
  • 5 = not checked locked frame.
  • 6 = checked locked frame.

There is also 1 sound properties added.

  • checkStateChangeSound

There is an example of this class in the examples/ui/controls/buttons folder.
