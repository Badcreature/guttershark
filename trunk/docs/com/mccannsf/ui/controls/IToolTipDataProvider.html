Interface IToolTipDataProvider
 
Packagecom.mccannsf.ui.controls
Interfacepublic interface IToolTipDataProvider
ImplementorsToolTipDataProvider

The IToolTipDataProvider interface allows for polymorphism with the data that is provided to an IToolTip instance.

An example of such data that might need to change per tooltip is data the specifies an icon to use, or a different color to use, etc.
