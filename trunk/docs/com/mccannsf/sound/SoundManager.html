Packagecom.mccannsf.sound
Classpublic class SoundManager
InheritanceSoundManager flash.events.EventDispatcher

The SoundManager class is used to control sounds in a flash movie through one interface. Actionscript 3 has a much different sound model from AS 2. This class is used to take care of those differences and bundle up a way to control sound globally.
